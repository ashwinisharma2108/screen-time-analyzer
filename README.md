Screen Time Analyzer

Overview
This project predicts a person's productivity level based on their daily habits such as screen time, sleep hours, and study hours using Machine Learning.

Features
- Takes user input (screen time, sleep, study)
- Predicts productivity level (Low, Medium, High)
- Provides suggestions for improvement
- Displays model accuracy and feature importance

Technologies Used
- Python
- Pandas
- Scikit-learn

How to Run
1. Install dependencies:
   python -m pip install pandas scikit-learn

2. Run the program:
   python main.py

Model Used
Decision Tree Classifier

Example
Input:
Screen Time: 7  
Sleep Hours: 4  
Study Hours: 1  

Output:
Productivity Level: LOW

Learning Outcome
- Understanding of Machine Learning workflow
- Data preprocessing and model training
- Real-world application of AI concepts
